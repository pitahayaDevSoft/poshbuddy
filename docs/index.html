¿Por qué usar PoshBuddy?

Segment Manager

Activa o desactiva segmentos (Git, Batería, Node) directamente en tu JSON sin romper la estructura. Edición quirúrgica garantizada.

One-Click Install

¿Ves un tema que te gusta en el catálogo oficial de Oh My Posh? Selecciónalo y PoshBuddy lo descarga, instala y aplica al instante.

Profile Shield

Gestionamos tu $PROFILE de PowerShell de forma no destructiva usando marcadores inteligentes. Nada se pierde, todo es reversible.

Instalación Rápida

git clone https://github.com/julesklord/poshbuddy.git
cd poshbuddy
cargo install --path .

Requiere Rust (Cargo) y Oh My Posh instalados en el sistema.
