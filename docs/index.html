architecture.txt

  • surgical json manipulation

    Powered by Serde, PoshBuddy performs precise edits to your OMP JSON files. Toggle segments without breaking custom styles.

  • async themeasset engine

    Built with Tokio for non-blocking theme discovery. Access 240+ themes with environment-isolated previews.

  • safe profile injection

    Uses a marker-based system to manage your $PROFILE. Reversible, auditable, and won't touch your existing logic.

  • responsive tui engine

    Dynamic 50/50 layout that adapts to your terminal size. Optimized for ultra-wide and mobile-style terminal views.

PoshBuddy Manual Section 1: UI & Operations

Synopsis

poshbuddy [OPTION]...
A high-density terminal user interface designed to streamline theme customization and shell segment manipulation.

Visual Demonstration

Global Keybindings

KeyAction Description
1Themes Explorer — Browse 240+ official remote themes.
2Font Manager — One-click Nerd Fonts installation.
3Segment Manager — Toggle theme components in real-time.
BBackup Manager — Manage and restore profile snapshots.
DDiagnostics — Check for fonts, shell, and compatibility.
H / EscReturn to Dashboard / Home screen.
QExit the application safely.

CLI Direct Access (Headless Mode)

For quick operations without launching the TUI, use the following command structure:

CommandPurpose
poshbuddy set theme <name>Search and apply a theme instantly.
poshbuddy install font <name>Atomic installation of a specific Nerd Font.
poshbuddy list themes [--local | --remote]Formatted table of all available configurations.
poshbuddy list fontsEnumerate all supported Nerd Fonts.

Operational Procedures

1. Fresh Environment Setup
Run System Diagnostic [D] first. If Oh My Posh is missing, PoshBuddy will initiate an atomic installation pipeline. Once verified, move to Font Manager [2] to install a Nerd Font (Windows Terminal recommended).

2. Surgical Theme Discovery
Navigate Themes Explorer [1]. Start typing to filter. PoshBuddy uses env_clear to ensure previews are unaffected by your current shell state. Apply with [ENTER].

3. Resilience & Security
- 10s Network Guard: Tokio aborts hung GitHub requests automatically.
- Marker Integrity: All profile changes are wrapped in # PoshBuddy tags for 100% reversible operations.
