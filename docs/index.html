The PoshBuddy Philosophy

Surgical Precision

Unlike standard theme managers, PoshBuddy performs precise JSON edits on existing theme files. Toggle segments like Git, Node, or Battery metrics without losing your custom formatting.

Unified Experience

The ThemeAsset engine bridges local configuration with the entire official Oh My Posh collection. Discover, preview, and deploy themes directly from the TUI interface.

Environment Integrity

Safety is not optional. Your PowerShell $PROFILE is managed using an injection system based on secure markers, ensuring that every modification is auditable and reversible.

Getting Started

Follow these steps to compile PoshBuddy for your system.

git clone https://github.com/julesklord/poshbuddy.git
cd poshbuddy
cargo install --path .

Requires Rust toolchain and Oh My Posh binary on system PATH.
