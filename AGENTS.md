# AGENTS.md - poshbuddy

## Project Overview
PoshBuddy is a Rust-based CLI tool and plugin installer. It includes robust cross-platform components, utilizing Cargo for builds and Playwright for UI/integration testing.

## Build and Test Commands
- **Build:** cargo build
- **Test (Rust):** cargo test
- **Test (Playwright):** 
px playwright test
- **Linting:** cargo clippy

## Conventions
- Follow standard Rust idioms and use ustfmt for formatting.
- Ensure cross-platform compatibility.
