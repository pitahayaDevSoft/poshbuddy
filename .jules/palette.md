## 2024-04-11 - Explicit Search State in TUIs
**Learning:** In Terminal User Interfaces (TUIs) like PoshBuddy, implicit states such as active search queries or filters must be explicitly rendered in the UI (e.g., dynamically updated block titles) to prevent confusing hidden state issues.
**Action:** Ensure active filters or search terms are visibly displayed in the relevant block titles or status lines.
